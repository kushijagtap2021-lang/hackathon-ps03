# 🌱 Which Crop Should I Grow? Finder

A hackathon-ready interactive hydroponic crop recommendation platform built for WebX × Farmspherica Innovations.

## 🚀 Features

### Quiz System

* Multi-step crop recommendation quiz
* Clickable option cards
* Animated transitions
* Progress tracking
* Eco Score system

### Smart Recommendations

* Crop matching based on:

  * Available Space
  * Light Conditions
  * Experience Level
  * Plant Care Availability
* JavaScript-based recommendation engine
* Dynamic crop filtering

### Premium UI

* Glassmorphism design
* Floating leaves animation
* Animated hero section
* Typing title effect
* Hover interactions
* Smooth transitions
* Responsive layout

### Visual Effects

* Confetti celebration
* Plant growth animation
* Parallax background
* Floating eco particles
* Result reveal animations

### User Experience

* Dark / Light mode
* Mobile responsive
* Start Over functionality
* Interactive crop cards

## 🌿 Crop Database

The application contains a hydroponic crop database including:

* Lettuce
* Basil
* Spinach
* Mint
* Kale
* Strawberries
* Cherry Tomatoes
* Cilantro

Each crop contains:

* Name
* Image
* Recommended Space
* Light Requirements
* Experience Level
* Recommendation Reason
